# AI Clinic Receptionist

This project is an AI-powered receptionist built using Retell AI and n8n.

## Features
- Patient verification
- Appointment scheduling
- Google Sheets patient database
- Cal.com calendar booking

## Tech Stack
- Retell AI
- n8n
- Google Sheets
- Cal.com
- Webhooks

## Workflow

User → Retell AI → n8n Webhook → Google Sheets → Calendar Booking
